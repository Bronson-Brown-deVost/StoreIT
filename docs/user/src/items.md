# Items

Items are the individual things you're tracking — tools, supplies, documents, collectibles, or anything else.

## Adding Items

### With AI Identification

1. Navigate to where the item is stored
2. Click **Add Item**
3. Take a photo or select one from your gallery
4. AI analyzes the photo and suggests a name, description, and tags
5. Review, adjust if needed, and click **Create**

### Manual Entry

1. Go to the **Add Item** page
2. Enter name, description, and tags manually
3. Select the parent location or container
4. Optionally attach a photo
5. Click **Create**

## Editing Items

Click the **Edit** button on any item's detail page to modify its name, description, or tags.

## Moving Items

1. Open the item's detail page
2. Click the **Move** button
3. Select the new location or container
4. Confirm the move

## Deleting Items

On the item's detail page, click **Delete Item** and confirm. This action cannot be undone.

## Item Details

Each item page shows:
- **Name and description**
- **Tags** — searchable keywords (often AI-generated)
- **Location path** — full breadcrumb trail showing where the item is stored
- **Photos** — all attached images
