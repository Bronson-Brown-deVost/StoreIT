# First Steps

After starting StoreIT, open `http://localhost:8080` in your browser (or your server's address).

## Create Your First Location

1. You'll see an empty home page with an **Add Location** button
2. Click it, enter a name (e.g., "Kitchen", "Garage", "Office")
3. Optionally add a description and photo
4. Click **Create**

Your location appears immediately.

## Add a Container

1. Navigate into a location by clicking on it
2. Click the **+** button and select **Add Container**
3. Name it (e.g., "Top Shelf", "Drawer Left", "Blue Box")
4. Click **Create**

Containers can be nested — open a container and add sub-containers the same way.

## Add an Item

There are two ways to add items:

### Quick Add (with AI)

1. Navigate to the location or container where the item lives
2. Click **Add Item**
3. Take a photo or select one from your gallery
4. The AI identifies the item and suggests a name, description, and tags
5. Review and adjust the suggestions, then click **Create**

### Manual Add

1. Go to `/add` or use the bottom navigation
2. Enter the item name and description manually
3. Select where it's stored
4. Click **Create**

## Search for Items

Use the search bar at the top of any page. StoreIT searches across item names, descriptions, and AI-generated tags. Results show the full location path so you know exactly where to find things.

## Install as an App

StoreIT is a Progressive Web App. On your phone:

- **iOS**: Open in Safari → tap Share → "Add to Home Screen"
- **Android**: Open in Chrome → tap the install prompt or Menu → "Install app"

This gives you a full-screen app experience with offline support.
